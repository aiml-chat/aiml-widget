AIML widget · local test harness

Loads ./dist/widget.js against your local API to verify recommendations and source citations end-to-end.

What to verify
  • #4 Recommendations: open the chat bubble (bottom corner). The welcome screen should show suggested-question chips. With no manual ones set, these auto-fill from your site's real top answered questions (needs prior chat history).
  • #5 Sources: ask a question that has indexed content. After the answer streams, a “Sources” block with clickable links should appear under it.
  • No content indexed yet → you'll get a “no answer” state (and a lead form if a Website ID is set). Index a site first via the dashboard.
